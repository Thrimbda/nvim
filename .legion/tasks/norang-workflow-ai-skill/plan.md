# norang-workflow-ai-skill

## 目标

将现有 Norang workflow 实践沉淀为面向 AI 的可复用 Skill，并生成可直接用于 PR 的交付文档。

## 要点

- 复用 docs/orgmode-norang-workflow.md 的既有规则作为事实来源
- 输出 AI 可执行流程（前置检查、日常操作、异常处理、回滚）
- 将 Skill 落地到项目根目录以便版本化协作
- 补齐 test/review/report/pr-body 交付链路

## 范围

- skills/norang-workflow/**
- .legion/tasks/norang-workflow-ai-skill/**
- docs/orgmode-norang-workflow.md
